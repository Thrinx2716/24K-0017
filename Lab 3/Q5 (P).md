## *PSEUDOCODE*

START

PRINT "Enter Principal amount (should be between 100 Rs to 1,000,000 Rs): "\
INPUT P

PRINT "Enter Rate of Interest (should be between 5% to 10%): "\
INPUT R

PRINT "Enter Time period (between 1 to 10 years): "\
INPUT T

SET I TO (P * R * T) / 100\
PRINT "The Interest is: ", I, " Rs"

END
